RalphDex Dashboard — hi-fi design pass

What this is: a rethink of the main Ralphdex webview focused on one-glance health, task graph + dependencies, and failure recovery. Everything runs inside a simulated VS Code frame to show it in context.

Core idea: a Mode toggle (Simple · Standard · Advanced) reshapes the surface for novices vs. power users, backed by savable Presets — matching the existing operatorMode setting. A plain-English "Now" hero tells beginners what's happening; Standard adds the real phase tracker; Advanced shows every lane.

  • Hero "Now" card — plain English status + a four-metric health strip. Always visible.
  • Task graph — real SVG graph with parent-child (dashed amber) and depends-on (solid) edges. Click a node to drill in.
  • Failure panel — moved up, not hidden in a tab. Category + confidence + summary + one-click "Auto-recover".
  • Agent lanes — each concurrent role on its own track with live phase tracker.
  • Iteration timeline — single-line rows, color-coded classification, clickable artifacts.

Try it: Switch modes in the sidebar. Click Start loop / Stop loop in the hero. Open the Tweaks toolbar toggle to swap theme, accent, density, and demo state.
